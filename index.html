📖 Flashing Guide

⚠️ Warning: Backup your data before flashing!

  1. Reboot to Custom Recovery (TWRP / OrangeFox).
  2. Wipe Data, Cache, System, Vendor and Dalvik Cache.
  3. Flash the ROM zip file.
  4. Flash the GApps zip file (if Vanilla build).
  5. Format Data (if changing ROM from FBE base).
  6. Reboot System and enjoy!
